Implementation Guide

1. Basic HubSpot Form Embed

<script charset="utf-8" type="text/javascript" src="//js.hsforms.net/forms/embed/v2.js"></script> <script> hbspt.forms.create({ region: "na1", portalId: "YOUR_PORTAL_ID", formId: "YOUR_FORM_ID", target: '#hubspot-form-container' }); </script>

2. Form with Custom Styling and Callbacks

<script> hbspt.forms.create({ region: "na1", portalId: "YOUR_PORTAL_ID", formId: "YOUR_FORM_ID", target: '#hubspot-form-container', css: '', onFormReady: function($form) { // Form is ready console.log('Form ready'); }, onFormSubmit: function($form) { // Form submitted console.log('Form submitted'); // Track with Google Analytics if (typeof gtag !== 'undefined') { gtag('event', 'form_submit', { 'event_category': 'engagement', 'event_label': 'webinar_registration' }); } }, onFormSubmitted: function($form) { // Form submission successful console.log('Form submission successful'); // Redirect or show thank you message setTimeout(function() { window.location.href = '/thank-you'; }, 1000); } }); </script>

3. Getting Your HubSpot Form Details

To find your form details:
  1. Go to HubSpot → Marketing → Lead Capture → Forms
  2. Click on your form
  3. Click "Share" or "Embed"
  4. Copy the embed code - it contains your portal ID and form ID
  5. Your portal ID is your HubSpot account ID

4. Advanced: Form with Pre-filled Fields

<script> // Pre-fill form based on URL parameters or user data hbspt.forms.create({ region: "na1", portalId: "YOUR_PORTAL_ID", formId: "YOUR_FORM_ID", target: '#hubspot-form-container', onFormReady: function($form) { // Pre-fill email if available const urlParams = new URLSearchParams(window.location.search); const email = urlParams.get('email'); if (email) { $form.find('input[name="email"]').val(email); } } }); </script>
